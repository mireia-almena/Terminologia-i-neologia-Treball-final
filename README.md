# Terminologia i neologia: Treball final

Aquest projecte té l’objectiu d’avaluar com els Models de Llenguatge Extensos generatius (LLMs) en català tracten la neologia, basant-se en l’enfocament de referència proposat per Zheng et al. (2024), anomenat NEO-BENCH. El projecte se centra en la seva tasca de Cloze Question Answering, mitjançant una anàlisi de sis models de llenguatge segons la seva capacitat per seleccionar neologismes en contextos reals.
